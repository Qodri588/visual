# Prompt

Buatkan aplikasi desktop modern bernama **Studio Visualizer** menggunakan framework dan library yang sesuai untuk performa tinggi. Aplikasi terinspirasi dari visualizer musik profesional dan memanfaatkan engine dari repository Studio:

[Studio GitHub Repository](https://github.com/hibuno/studio?utm_source=chatgpt.com)

Gunakan repository tersebut sebagai dasar rendering dan visual effects jika memungkinkan.

## Tujuan Aplikasi

Aplikasi digunakan untuk membuat video visualizer musik untuk YouTube, playlist musik, lagu nostalgia, lo-fi, podcast, dan konten audio lainnya.

Fokus pada kemudahan penggunaan, performa tinggi, preview real-time, dan rendering video berkualitas tinggi.

---

* UI modern dan responsif
